How to Use the Floating Icon

Follow these steps to implement the floating icon on your webpage:

  1. Include the CSS file in the `` section of your HTML: 
    <link rel="stylesheet" href="css/v1.0/floatingIcon.css">
  2. Include the JavaScript file before the closing `` tag: 
    <script src="js/v1.0/floatingIcon.js"></script>
  3. Call the `createFloatingIcon` function in your script with the desired options: 
    
createFloatingIcon({
  icon: "üîó", // The icon to display
  position: "bottom-left", // Position: bottom-left, bottom-right, top-left, or top-right
  tooltip: "Go to Google", // Tooltip text
  link: "https://www.google.com" // URL to open when clicked
});

Options

The `createFloatingIcon` function accepts the following options:

  • icon: The emoji or text to display as the icon (default: ‚ùì).
  • position: The position of the icon on the screen. Options: `bottom-right`, `bottom-left`, `top-right`, `top-left` (default: `bottom-right`).
  • link: The URL to open when the icon is clicked. If not provided, an alert will be shown.
  • tooltip: The text to display when hovering over the icon (default: "Click me!").
